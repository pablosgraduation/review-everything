# Contributing

## Prerequisites

- Rust 1.85+
- [difftastic](https://difftastic.wilfred.me.uk/) (`difft`) must be installed and on your `PATH`

## Build

```sh
cargo build
```

## Test

```sh
cargo test
```

## Lint

```sh
cargo clippy
```

## Submitting Changes

1. Fork the repo and create a feature branch
2. Make your changes
3. Run `cargo test` and `cargo clippy` before submitting
4. Open a pull request against `main`
